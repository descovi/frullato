frullato
========

From a folder with inside random name images to a folder with ordered name images.

```
folder/lorem.jpeg
folder/ipsum.jpeg 
```
became
```
folder/1.jpeg 
folder/2.jpeg
```

How?
----

```bash
ruby frullato.rb
```
and

```bash
"-------------------"
" Enter the folder: "
"-------------------"
```

And choose folder target (`img` for example)
